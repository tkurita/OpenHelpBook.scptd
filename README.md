OpenHelpBook
============

OpenHelpBook is an AppleScript Library to open a help book in a bundle with Help Viewer.

## Usage

http://www.script-factory.net/XModules/OpenHelpBook/manual/index.html

## Building

Reqirements :
* OS X 10.9 or later.
